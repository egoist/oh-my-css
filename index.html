Get started

Current version: 0.1.2 (Unstable)

Install

load Oh-My-CSS in your project

<!-- Oh My CSS core stylesheet -->
<link rel="stylesheet" href="//cdn.ohmycss.com/latest/css/ohmycss.css">

<!-- Oh My CSS lite version (without normalize.css) -->
<link rel="stylesheet" href="//cdn.ohmycss.com/latest/css/ohmycss.lite.css">

or Bower

bower install ohmycss --save
